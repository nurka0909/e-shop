📦 E-Commorce (Ecommerce Web / App)
🔖 Товч танилцуулга

E-Commorce бол online shop / e-commerce веб аппликейшн юм.
Энэ проект нь хэрэглэгчдэд бүтээгдэхүүн харуулах, сагсанд нэмэх, захиалга хийх, удирдагчдад бүтээгдэхүүн болон захиалгыг удирдах боломжийг олгодог.

🚀 Гол боломжууд

✔ Бүтээгдэхүүн жагсаалт
✔ Бүтээгдэхүүн дэлгэрэнгүй хуудас
✔ Search / Filter үйлдэл
✔ Хэрэглэгчийн данс / Auth
✔ Сагс / checkout
✔ Захиалга үүсгэх
✔ Admin dashboard (Product manage)
✔ Responsive дизайн

📁 Project Structure (Жишээ)

├── .gitignore

⚙️ Технологи: React + Vite + CSS/SCSS + React Router
Бусад боломжтой: Tailwind CSS, Context API/Redux, Firebase/Backend API

🧰 Хэрэглэсэн технологи

Frontend: React, Vite

Routing: react-router-dom

State management: React Context / Redux

Styling: CSS / SCSS / Tailwind

Backend / API: Firebase / Node.js / Express (хаа байгаа API-тай холбогдоно)
